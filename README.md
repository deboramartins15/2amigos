# 2amigos

Sistema para controlar notas fiscais recebidas por uma empresa de logística

## Frontend
### Instalar dependências
`npm i`

### Executar o projeto
`npm start`

## Backend
### Instalar dependências
`npm i`

### Executar o projeto
`npm run dev`

ATENÇÃO!!
Deve-se configurar as variáveis de ambientes antes de executar o projeto
